Usco-Iot Project

Ingreso

Modulo de entrenamiento para aplicaciones del Internet de las cosas (IoT)

El Internet de las cosas es una interacción entre el mundo físico y el digital, con el fin de fomentar la investigación y el aprendizaje sobre este tópico se diseñó un sistema que permita el entrenamiento y el desarrollo de habilidades para el desarrollo de proyectos IoT; el sistema se compone de un módulo físico denominado módulo-IoT y un módulo virtual, el módulo IoT se compone por una ESP32 WiFi, sensores DHT11 (Temperatura y humedad), MQ135 (Calidad del aire), HC-SR04 (Ultrasonido) y una etapa de potencia para conectar actuadores de alta potencia; los datos censados se enviarán a firebase y mediante el servicio de Real Time Database se almacenarán asignándoles un path dentro de la base de datos para posteriormente poder acceder a estos valores los cuales se mostrarán en un dashboard diseñado en HTML, CSS y JavaScript.
